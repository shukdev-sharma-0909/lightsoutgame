Process:

The goal was to create a Lights Out game using HTML, CSS, and jQuery. I started by setting up the HTML structure, including the grid container and control buttons to change grid size and reset the game. I then styled the grid cells with a transparent, card-like effect using rgba colors and box-shadow for a subtle depth.

Creative Choices:

I chose a transparent card effect for the grid cells to create a sleek, modern interface. The subtle box-shadow added a 3-dimensional appearance, enhancing the visual experience. Providing buttons to switch grid sizes (3x3, 4x4, and 5x5) made the game scalable and adaptable.

Challenges and Insights:

One challenge was accurately toggling neighboring cells, which I solved by calculating row and column indices with arithmetic operations. Maintaining a consistent visual appearance across different grid sizes required dynamic adjustments to CSS grid properties.

Through this project, I gained insights into efficient DOM manipulation with jQuery, CSS Grid layouts, and the importance of scalable design. I also learned that attention to small visual details, like shadows and rounded corners, significantly enhances the user experience and interface aesthetics.
